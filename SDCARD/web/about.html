Sobre o Projeto PrintSense

Monitoramento inteligente para qualidade em impressão 3D

📌 Sobre o Projeto

O PrintSense é um projeto desenvolvido para monitorar o ambiente de impressão 3D, garantindo que fatores críticos como temperatura, umidade, luminosidade e nível de ruído estejam sempre dentro dos padrões ideais.

A proposta do sistema é evitar falhas nas impressões, aumentar a durabilidade dos materiais e melhorar a qualidade final das peças produzidas. Com isso, o projeto contribui diretamente para um processo mais eficiente, seguro e automatizado.

⚙️ Como Funciona

O PrintSense utiliza sensores conectados a um microcontrolador que coleta dados ambientais em tempo real. Essas informações são enviadas para uma interface web, onde o usuário pode acompanhar gráficos, histórico e indicadores de qualidade.

Além disso, o sistema permite selecionar diferentes tipos de materiais (PLA, ABS, PETG, Resina), ajustando automaticamente os parâmetros ideais para cada situação.

🎯 Objetivo

O objetivo do projeto é proporcionar maior controle sobre o ambiente de impressão, reduzindo erros, otimizando recursos e garantindo maior qualidade nas peças produzidas. Além disso, o PrintSense também serve como uma aplicação prática de conhecimentos em IoT, desenvolvimento web e análise de dados.
