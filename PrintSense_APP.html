📋 Visão Geral do Projeto

PrintSense Desktop é uma aplicação profissional de monitoramento em tempo real para ambientes de impressão 3D, desenvolvida em C# WPF (.NET Framework 4.8).

O sistema integra sensores ESP32 via REST API, processamento de Machine Learning para análise preditiva, visualização avançada com gráficos em tempo real, e exportação de relatórios profissionais em Excel.

🔄 Fluxo de Dados

1. Inicialização
Carrega tema, configurações, calibrações e modelo ML. Inicializa gráficos e timer.
2. Conexão ESP32
HttpClient envia GET para http://IP:PORT/api/data a cada intervalo configurado (padrão: 1s).
3. Processamento
Deserializa JSON → Aplica calibração → Atualiza UI → Adiciona ao histórico → Atualiza gráficos.
4. Machine Learning
Se modelo treinado: faz predição em tempo real. Se não: permite treinar com dados históricos.
5. Análise Inteligente
Processa leituras, identifica padrões, calcula ROI e gera recomendações automatizadas.
6. Exportação
EPPlus gera Excel com sheets: Dados completos + Estatísticas + Distribuição de status.

🔍 Detalhes Técnicos Importantes

⚡ Otimizações Implementadas

  • Gráficos sem Flickering: UpdateCharts() usa .Clear() + .Add() ao invés de recriar ChartValues
  • HttpClient Reutilizável: Recriado apenas quando timeout muda (evita InvalidOperationException)
  • Async/Await Pattern: Todas operações HTTP são assíncronas (sem travar UI)
  • Null Checks: Validações em LoadSettings, RunAdvancedAnalysis e métodos de atualização de UI
  • Dynamic Resources: Cores do tema usam {DynamicResource} para mudança em tempo real

🐛 Correções Críticas Realizadas

  • Ambiguidade System.Drawing.Brushes vs System.Windows.Media.Brushes: Resolvido com namespace completo
  • HttpClient.Timeout após primeira requisição: Implementado dispose + recreate pattern
  • Average() em sequência vazia: Adicionado checks antes de LINQ aggregations
  • NullReferenceException em LoadSettings: _updateTimer criado antes de LoadSettings()
  • Gráficos piscando: Mudado de recreate para update dos ChartValues existentes

📑 Funcionalidades Detalhadas por Tab

📊 Tab 1: Dashboard

  • 3 Gráficos LiveCharts: Temperatura, Umidade e Som (últimas 20 leituras)
  • Cards de Sensores: Valores atuais + ranges ideais + status (OK/Warning/Error)
  • Indicador LED: Verde/Amarelo/Vermelho sincronizado com ESP32
  • Status Geral: IDEAL (verde) | BOM (laranja) | RUIM (vermelho) com descrição
  • Atualização em Tempo Real: Timer configurável (padrão: 1 segundo)
  • Seleção de Material: PLA, PETG, ABS, Resina com thresholds específicos

🔧 Tab 2: Calibração

  • Sliders de Offset: Temperatura (±5°C) e Umidade (±20%)
  • Preview em Tempo Real: Mostra valores calibrados vs. brutos
  • Persistência JSON: Salva em calibration.json automaticamente
  • Reset Rápido: Botão para zerar todos os offsets
  • Aplicação Imediata: Calibrações aplicadas em todas as leituras subsequentes

⚙️ Tab 3: Configurações Avançadas

  • Conexão ESP32: IP, Porta, Timeout (com validação)
  • Intervalo de Atualização: Slider 0.5s - 10s
  • Teste de Conexão: Botão para validar antes de salvar
  • Persistência Automática: Salva em settings.json
  • Reconexão Inteligente: Desconecta automaticamente se IP/Porta mudarem

📋 Tab 4: Relatórios

  • Estatísticas Completas: Min/Avg/Max para cada sensor
  • Distribuição de Status: Contagem e % de IDEAL/BOM/RUIM
  • Exportação Excel (EPPlus): 2 sheets (Dados + Estatísticas)
  • Formatação Profissional: Headers azuis, bordas, auto-width
  • Timestamp Completo: Todas leituras com data/hora

🧠 Tab 5: Análise Inteligente

  • Treinamento ML.NET FastTree: 100 árvores, 20 folhas
  • Features: Temperatura, Umidade, Luz, Som
  • Label: Binary (IDEAL/BOM = true, RUIM = false)
  • Predição em Tempo Real: Avalia cada leitura com confiança %
  • Importância de Features: Barras visuais (Temp > Hum > Som > Luz)
  • Análise Avançada: Identifica melhor horário, calcula ROI de equipamentos
  • Recomendações Automatizadas: Ar-condicionado, Desumidificador, Isolamento Acústico
  • Taxa de Sucesso por Material: PLA, PETG, ABS com barras de progresso

✨ Conclusão

O PrintSense Desktop App representa uma solução completa e profissional para monitoramento de ambientes de impressão 3D, combinando tecnologias modernas como WPF, ML.NET, e LiveCharts com uma arquitetura robusta e escalável.


A aplicação demonstra boas práticas de desenvolvimento .NET, incluindo async/await pattern, data binding, MVVM principles, dependency management, error handling, e persistência de dados. O sistema de temas com Dark Mode cyberpunk adiciona um diferencial visual único, enquanto a análise de Machine Learning proporciona insights preditivos valiosos.
